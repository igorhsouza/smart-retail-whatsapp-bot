## Overview

Automação de atendimento via WhatsApp com IA, focada em interpretar mensagens, organizar contexto e direcionar o cliente com base em lógica de negócio.

---

## Capabilities

- interpretação de mensagens em tempo real  
- classificação de intenção  
- filtragem de mensagens irrelevantes  
- resposta automatizada com contexto  
- direcionamento para produtos ou atendimento humano  
- manutenção de histórico por cliente  

---

## Stack

- n8n  
- OpenAI  
- WAHA API  
- Supabase  
- Google Drive  

---

## Core Features

- atendimento automatizado  
- classificação inteligente  
- roteamento de mensagens  
- memória de contexto  
- transcrição de áudio  
- leitura de arquivos (PDF / imagem)  
- fallback para atendimento humano
